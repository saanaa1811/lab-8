Tim Berners Lee: The Inventor of the World Wide Web

Tim Berners Lee created the World Wide Web in 1989, changing the way we access and exchange information all through the world. His invention of the web has been one of the most important contributions to the technological age.

The Three Key Parts of the World Wide Web Invention

1) HTML (HyperText Markup Language)
A standardized approach for arranging online pages with markup tags. HTML determines the layout of web pages with features such as headers, paragraphs, and hyperlinks..
2) HTTP (HyperText Transfer Protocol)
The protocol used to send information between a web browser and a web server. It allows for web-based communication and resource access.
3) URL (Uniform Resource Locator)
A particular URL for locating web resources such as webpages or photos. A URL allows users to get to a particular page on the World Wide Web.
